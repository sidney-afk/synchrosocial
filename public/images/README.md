# Images

Drop real image files here, then reference them as `/images/<filename>` in the
relevant page (see the main `README.md` → "Swapping in real images").

Until a real file is provided, the site shows tasteful placeholders, so it never
looks broken.

- Square images work best for the circular avatars (e.g. 400×400).
- Keep files reasonably small (web-optimized JP/WebP) for fast loading.
